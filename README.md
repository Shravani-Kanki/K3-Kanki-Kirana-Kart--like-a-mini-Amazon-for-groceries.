K3 KankiKiranaKart — Online Grocery Store

Welcome to KankiKiranaKart, a modern online grocery shopping platform inspired by real kirana stores.  
This project combines a powerful **Spring Boot backend** with a clean and responsive frontend to create a smooth user experience.



>Current Features
-  User Login  
-  User Signup  
-  Forgot Password System  
-  Fully designed HTML/CSS/JS frontend  



>Upcoming Enhancements
-  Product Listing & Categories  
-  Shopping Cart   
-  Secure Payment Integration  
-  Admin Dashboard (Add/Edit/Delete Products)  
-  Sales & Analytics (Admin)  



>Tech Stack
 **Backend**
- Spring Boot  
- Spring MVC  
- Spring Data JPA / Hibernate  
- MySQL Database  

 **Frontend**
- HTML  
- CSS  
- JavaScript  



>Objective
To build a complete mini-ecommerce platform with full user authentication, real-time product management, and a seamless shopping flow — perfect for learning Full Stack Development, Spring Boot, and REST APIs.



>Project Status
Actively in Development — new modules being added.
